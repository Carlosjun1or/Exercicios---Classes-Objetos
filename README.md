## 📚 Lista Classes e Objetos — Exercícios

Projeto desenvolvido para praticar conceitos básicos de **Programação Orientada a Objetos (POO)** em C++, utilizando **classes, construtores e métodos**.

---

## 📋 Descrição

Este projeto contém a resolução de 3 exercícios propostos em sala, com foco na criação de classes e uso de objetos.

Os exercícios abordam:

- Operações matemáticas
- Verificação e classificação de triângulos
- Cálculo de valor em estacionamento

O programa possui um **menu interativo com switch**, permitindo escolher qual exercício executar.

---

## ⚙️ Funcionalidades

### 🔹 Exercício 1 — Classe Matemática

Criar uma classe chamada **Matematica** que contenha:

- Dois atributos da classe:
  - A (inteiro)
  - B (inteiro)

- Métodos:
  - Entrada de dados
  - Soma
  - Multiplicação
  - Subtração
  - Divisão

**Observação:**  
Realizar o tratamento da divisão, não permitindo divisão por zero.

---

### 🔹 Exercício 2 — Classe Triângulo

Criar uma classe chamada **Triangulo** que contenha:

- Variáveis da classe representando os três lados do triângulo

- Métodos:
  1. Verificar se os valores formam um triângulo
  2. Identificar o tipo de triângulo

Tipos de triângulo:

- Escaleno → três lados diferentes  
- Isósceles → dois lados iguais  
- Equilátero → três lados iguais  

**Observação:**  
Antes de identificar o tipo, é obrigatório verificar se os valores realmente formam um triângulo.

---

### 🔹 Exercício 3 — Classe Estacionamento

Criar uma classe chamada **Estacionamento** que contenha:

- Atributos:
  - Dia
  - Hora de entrada
  - Hora de saída

- Métodos:
  1. Solicitar os dados ao usuário
  2. Calcular o valor a ser pago

**Regra:**  
Cada hora no estacionamento equivale a **R$ 5,00**.
